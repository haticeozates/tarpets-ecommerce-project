A full-stack e-commerce application designed for pet owners, featuring a modern React frontend and a robust Spring Boot backend.

## Features

- **Product Management:** Filtered product listing by categories (Cat, Dog, Bird, Fish, etc.) and subcategories.
- **User Authentication:** Secure registration and login system using JWT (JSON Web Token).
- **Pet Management:** Users can register their pets and manage pet-specific information.
- **Shopping Cart & Orders:** Fully functional cart system with order history tracking.
- **Payment Integration:** Secure checkout process integrated with Stripe API.
- **Admin Dashboard:** Special interface for admins to manage users and products.

## Tech Stack

**Frontend:**
- React.js
- Axios (API Communication)
- Bootstrap (UI Components)
- Context API (State Management)

**Backend:**
- Java & Spring Boot
- Spring Security & JWT
- Spring Data JPA (Hibernate)
- MySQL / H2 Database

**Integrations:**
- Stripe API (Payment Gateway)

## Project Structure
- `SourceCode/frontend`: React application files.
- `SourceCode/backend`: Spring Boot API and business logic.
- `SourceCode/database`: SQL scripts and database schema.
